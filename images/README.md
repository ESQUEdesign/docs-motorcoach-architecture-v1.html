# Hero coach images

Drop ten top-view motorcoach photos here, named `coach-01.png` … `coach-10.png`
(`.jpg`/`.webp` also fine — update the paths in `hero.html` to match).

Guidelines for best results:
- **Top-down view**, shot from directly above.
- **Nose at the top** of the frame (the hero rotates each into its clock slot).
- **Transparent PNG** (or clean white background) so coaches sit on the white hero.
- Roughly portrait/tall crop (~1:3.8) framed tight to the coach.

To reuse a single photo for all ten, set `DEFAULT_IMAGE` in `hero.html`.
Any slot whose image is missing falls back to the built-in vector coach.
